Практична робота №3

Тема: Робота з текстовими та табличними документами у редакторі Word

Мета роботи: навчитися працювати з текстовими та табличними документами у Microsoft Word.

Хід роботи:

  1. Завантажити текстовий редактор Microsoft Word.
  2. Створити файл «Перелік елементів».
  3. Ввести таблицю та графічні об єкти.
  4. Завантажити текстовий редактор Microsoft Word.
  5. Створити файл «Організація захищеного режиму».
  6. Ввести текст та графічні об єкти.
  7. Завантажити текстовий редактор Microsoft Word.
  8. Створити файл «Обробка керування».
  9. Ввести текст та графічні об єкти.
