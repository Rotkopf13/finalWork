Практична робота №8

Тема: Обробка даних в Microsoft Access. Створення таблиць та зв язків.

Мета роботи: навчитися створювати таблиці та зв язки між таблицями у Microsoft Access 2007.

Хід роботи:

  1. Завантажити базу даних Microsoft Access 2007.
  2. Створити файл «Бібліотека».
  3. Створити таблиці.
  4. Створити зв язок між таблицями.
